# Rating Color Script

Скрипт для расширения функциональности медиаплеера Lampa с цветной стилизацией рейтинга, статуса и качества, а также добавлением удобных кнопок управления.

## Описание

**ratingColor_New.js** - модуль для Lampa, который добавляет:

### Функциональность

#### 1. Цветные плашки рейтинга
- Стилизирует плашки с рейтингом согласно значению:
  - **0.0 - 3.9**: Синий градиент (`#0F6775` → `#0F5375`)
  - **4.0 - 6.9**: Оранжевый градиент (`#FF8707` → `#FFC107`)
  - **7.0 - 7.9**: Зелёный градиент (`#43A047` → `#8CFF00`)
  - **8.0 - 10.0**: Жёлтый градиент (золотистый)

#### 2. Плашка качества
- Жёлтая стилизованная плашка для отображения качества видео

#### 3. Плашка статуса
- Градиентная плашка для отображения статуса контента

#### 4. Кнопка выхода
- Добавляет кнопку выхода в верхнее меню
- Кроссплатформенная поддержка:
  - Apple TV
  - Tizen (Samsung TV)
  - WebOS (LG TV)
  - Android
  - Orsay
  - NW.js

#### 5. Кнопка перезагрузки
- Добавляет кнопку перезагрузки в верхнее меню
- Быстро перезагружает приложение

## Настройки

В приложении Lampa доступны следующие параметры в разделе "Fovway":

| Параметр | Тип | Описание | По умолчанию |
|----------|-----|---------|--------------|
| Цветные плашки | Триггер | Включает/выключает цветные плашки на рейтинг, статус, качество | Включено |
| Кнопка выход | Триггер | Кнопка выхода в верхнем меню | Включена |
| Кнопка перезагрузки | Триггер | Кнопка перезагрузки в верхнем меню | Включена |

## Установка

1. Добавьте скрипт в ваш проект Lampa
2. Загрузите его как модуль расширения
3. Скрипт автоматически инициализируется при запуске приложения

## Технические детали

- **IIFE паттерн**: Скрипт использует самовызывающуюся функцию для изоляции кода
- **MutationObserver**: Отслеживает изменения DOM для динамического применения стилей
- **Cross-platform**: Поддерживает различные платформы и устройства
- **Storage API**: Использует хранилище Lampa для сохранения настроек

## Совместимость

Требует подключённой библиотеки **Lampa** с поддержкой:
- SettingsApi
- Storage API
- Listener API
- Platform detection
